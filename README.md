## Stock-Analysis Dashboard
Interactive dashboard for analyzing historical stock prices of major tech companies using Streamlit, Plotly, and Pandas.

# Overview
This project focuses on time-series analysis of stocks (AAPL, AMZN, GOOG, MSFT) using 5-year historical datasets.
It includes:

* Closing price trends
* Daily returns
* Moving averages (10, 150, 200 days)
* Resampling (Monthly, Quarterly, Yearly)
* Correlation heatmap between stocks

Built entirely in Python with an interactive Streamlit UI.

# Features
1. Closing Price Over Time
Interactive Plotly line charts showing price movement.

2. Moving Averages
Visualizes long- and short-term market trends.

3. Daily Returns (%)
Shows volatility and the stock's day-to-day performance.

4. Resampled Trends
Average close price:
Monthly
Quarterly
Yearly

5. Correlation Heatmap
Compares relationships between top tech stocks.
